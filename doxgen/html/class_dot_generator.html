Detailed Description

Генерирует DOT-диаграмму классов по результатам разбора Java-кода.

Формирует файл structure.dot в указанном выходном каталоге. Каждый класс представляется узлом-таблицей с полями, методами и конструкторами. Связи между классами отражают наследование, реализацию интерфейсов и вложенность.
